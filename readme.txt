Στο αρχείο myplaylist βρίσκονται 4 τραγούδια με τις εξής ονομασίες: short1.mp3,short2.mp3,short3.mp3,short4.mp3 .
Επίσης υπάρχει και αρχείο .m3u με τα συγκεκριμένα κομμάτια για να κληθεί απο το πρόγραμμα.  
Το πρόγραμμα λειτουργεί όπως ζητήθηκε με την εντολή 


java -jar target/MyJukebox-0.0.1-SNAPSHOT-jar-with-dependencies.jar <song/best.m3u>  <optional strategy>

~Εννοείται πως οι εντολές order και random για να έχουν λογική μπορούν να κληθούν μονο με το αρχειο m3u~
